ЧТО НУЖНО СДЕЛАТЬ:
------------------

1. ЗАКОММИТИТЬ И ЗАПУШИТЬ:

   git add .
   git commit -m "Fix: Add SSL certificate for Timeweb PostgreSQL"
   git push

2. ПРОВЕРИТЬ DATABASE_URL В APP PLATFORM:

   Таймвеб → App Platform → keykurs.ru → Настройки

   DATABASE_URL должен быть БЕЗ ?sslmode=verify-full

   ✅ Правильно:
   postgresql://gen_user:PASSWORD@...twc1.net:5432/default_db

   ❌ Неправильно:
   postgresql://gen_user:PASSWORD@...twc1.net:5432/default_db?sslmode=verify-full

3. ДОЖДАТЬСЯ АВТОДЕПЛОЯ (3-5 минут)

4. ПРОВЕРИТЬ РАБОТУ:

   https://keykurs.ru/health
   → Должно быть: {"status":"ok"}

   https://keykurs.ru/api/db-check
   → Должно быть: {"status":"connected"}

5. ПРИМЕНИТЬ SQL МИГРАЦИИ:

   Таймвеб → Базы данных → default_db → SQL редактор

   Применить по порядку:
   - timeweb-migrations/01_create_auth_system.sql
   - timeweb-migrations/02_create_platform_schema.sql
   - timeweb-migrations/03_setup_rls_policies.sql
   - timeweb-migrations/04_add_additional_features.sql
