# Команды запуска

## Локальная разработка

### Установка (первый раз)

```bash
# Frontend зависимости
npm install

# Backend зависимости
cd backend-api && npm install && cd ..
```

### Запуск в разных терминалах

**Terminal 1 - Backend API:**
```bash
cd backend-api
npm start
```
→ `http://localhost:3000`

**Terminal 2 - Frontend:**
```bash
npm run dev
```
→ `http://localhost:5173`

### Проверка подключения

```bash
# Health check
curl http://localhost:3000/health

# Проверка БД
curl http://localhost:3000/api/db-check

# Список таблиц
curl http://localhost:3000/api/db-tables
```

## Production (Timeweb)

### 1. Собрать Frontend

```bash
npm run build
```

Результат в папке `build/` → загрузить в Timeweb Apps (Static Website)

### 2. Развернуть Backend API

Загрузить папку `backend-api/` в Timeweb Apps (Backend Node.js)

### 3. Добавить переменные окружения

В Settings App добавить:
```
DATABASE_URL=postgresql://user:pass@host:5432/db?sslmode=require
JWT_SECRET=your_secret
ALLOWED_ORIGINS=https://keykurs.ru,https://www.keykurs.ru
PORT=8000
NODE_ENV=production
```

### 4. Привязать домены

- Frontend: `keykurs.ru` → Static Website
- Backend: `api.keykurs.ru` → Backend App

### 5. Проверить

```bash
curl https://api.keykurs.ru/health
curl https://api.keykurs.ru/api/db-check
```

## Быстрый старт

```bash
# Клонировать/распаковать проект
cd project

# Установить все зависимости
npm install && cd backend-api && npm install && cd ..

# Terminal 1: Backend
cd backend-api && npm start

# Terminal 2: Frontend
npm run dev

# В браузере
# Frontend: http://localhost:5173
# API: http://localhost:3000/health
```
