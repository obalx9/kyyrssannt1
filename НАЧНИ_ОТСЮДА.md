### Шаг 2: Проверь настройки на Таймвеб

Зайди в панель Таймвеб → App Platform → твое приложение

**ВАЖНО:** Убери `?sslmode=verify-full` из DATABASE_URL

**Было:**
```
DATABASE_URL=postgresql://...@...twc1.net:5432/default_db?sslmode=verify-full
```

**Должно быть:**
```
DATABASE_URL=postgresql://...@...twc1.net:5432/default_db
```
