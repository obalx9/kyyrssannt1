================================================================================
PROJECT STRUCTURE
================================================================================

Multi-module application:
  - Frontend: React + TypeScript + Vite (optimized: 468KB gzipped)
  - Backend: Express.js API server
  - Database: PostgreSQL (Timeweb hosted)
  - Storage: S3-compatible (Timeweb S3)

================================================================================
DEPLOYMENT OPTIONS
================================================================================

OPTION 1: DOCKER (Recommended for Timeweb Container)
  1. Build: docker build -t kursat-app:latest .
  2. Push to registry
  3. Deploy to Timeweb Container
  4. Health check will run automatically on port 3000

OPTION 2: TRADITIONAL VM (with PM2)
  1. Run: bash ./deploy.sh
  2. Application starts with PM2
  3. Automatically restarts on failure
  4. Logs available via: pm2 logs kursat-api

================================================================================
PRE-DEPLOYMENT VERIFICATION

================================================================================
TROUBLESHOOTING
================================================================================

Container won't start?
  1. Run: bash ./pre-deploy-check.sh
  2. Check logs: docker logs <id> or pm2 logs kursat-api
  3. Verify DATABASE_URL format
  4. Ensure database is ready

Dependencies error?
  cd backend-api && npm ci && cd ..

Database connection failed?
  1. Verify DATABASE_URL
  2. Test: psql postgresql://...
  3. Check firewall

S3 errors?
  1. Verify S3_ACCESS_KEY, S3_SECRET_KEY
  2. Check S3_BUCKET name
  3. Test: curl https://s3.timeweb.com/
